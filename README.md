### 安裝步驟
1. 下載並安裝 [Node.js](https://nodejs.org/dist/v12.14.1/node-v12.14.1-x64.msi)
2. 下載專案 [Github](https://github.com/oscarada87/issue-tracking-system-frontend.git)
3. 進入到專案資料夾後開啟 PowerShell 或 CMD，按下列步驟進行部屬

    ```
    npm install
    ```
    ```
    npm run serve
    ```
4. 在瀏覽器輸入此連結即可使用本系統： 
    ```
    http://localhost:8080/
    ```

### 預設可用帳號
